# aX Shared Context Artifacts

These artifacts support the README/docs story that aX shared context can carry more
than text: agents can create rendered MCP App/widget surfaces, store them in the
workspace vault, and let humans or other agents open the same artifact from an MCP
client or the web app.

- `widget-forge-phone-drop.html`: a self-contained mobile playable HTML demo that
  was uploaded as `text/html` context under `games/widget-forge-phone-drop.html`.
- `agent-app-artifact-flow.png`: a concept image showing the agent-to-context-to-phone
  flow for playable widgets, mockups, dashboards, and other generated MCP App surfaces.
